# 换汇资金托管系统 · App 简介

> 一款面向小型换汇/钱庄业务的轻量级资金流转追踪工具，支持多币种托管、中转商管理与客户账单报表。

---

## 定位

本系统适用于以下场景：

- 代客户收取现金或接收转账，再换汇转出
- 需要追踪每笔资金从收入到转出的完整状态
- 管理多个中转商和多个公司账号的资金分配
- 向客户提供清晰的进出账流水记录

---

## 核心功能

| 模块      | 说明                      |
| ------- | ----------------------- |
| 👥 实体管理 | 客户、中转商、公司账号、客户目标账号的增删管理 |
| 📊 资金流程 | 创建现金订单与转账订单，可视化流程图      |
| 📋 订单管理 | 查看所有订单，按状态筛选与推进         |
| 💱 货币兑换 | 在公司账号间执行多币种兑换，记录历史      |
| 🏦 账号余额 | 实时查看各公司账号余额             |
| 📝 客户流水 | 每位客户的完整进出账明细与汇总报表       |
| ⚙️ 汇率设置 | 设置 AUD 基准汇率             |

---

## 支持货币

`AUD` · `USD` · `CNY` · `EUR`

---

## 技术说明

- **技术栈**：纯前端 HTML + CSS + JavaScript，无需服务器
- **数据存储**：浏览器 LocalStorage（本地，无云同步）
- **部署方式**：静态网页，可托管到任意静态服务器

> ⚠️ 数据存储在浏览器本地，清缓存或换浏览器会丢失数据，请定期备份。
