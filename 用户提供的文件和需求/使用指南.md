# 使用指南

> 换汇资金托管系统操作手册 · 支持货币：AUD / USD / CNY / EUR

---

## 一、初始配置

首次使用前，需先在 **👥 实体管理** 页面完成以下配置：

### 1. 添加客户

填写客户姓名和联系方式（电话/微信），点击 **添加客户**。

### 2. 添加中转商 (其实中转账账号跟普通账号也是一样的.)

填写中转商名称（如 Bob、Gqc、PIN）和联系方式，点击 **添加中转商**。

### 3. 添加公司账号

填写账号名称（如 HSBC AUD账号）、银行名称、账号和主货币，点击 **添加公司账号**。

### 4. 添加客户目标账号

填写所属客户、账号名称、银行、账号、货币和持有人姓名，点击 **添加目标账号**。

---

## 二、日常操作

### 创建现金订单

适用于客户以现金形式存款，经中转商转入公司账号的流程。

1. 进入 **📊 资金流程** → 点击 **💵 现金流程** 的"创建现金订单"
2. 填写：客户、现金金额、货币、中转商、目标货币、客户目标账号
3. 点击 **创建订单**

**状态流转：**

```
待处理 → 交中转商 → 在公司账号 → 已完成
```

| 当前状态  | 下一步操作     |
| ----- | --------- |
| 待处理   | 点击"交给中转商" |
| 交中转商  | 点击"确认到账"  |
| 在公司账号 | 点击"完成"    |

---

### 创建转账订单

适用于客户直接银行转账到公司账号的流程。

1. 进入 **📊 资金流程** → 点击 **🏦 直接转账流程** 的"创建转账订单"
2. 填写：客户、转账金额、货币、收款公司账号、目标货币、客户目标账号
3. 点击 **创建订单**

**状态流转：**

```
在公司账号 → 已完成
```

---

### 执行货币兑换

1. 进入 **💱 货币兑换** 页面
2. 选择公司账号、源货币及数量
3. 设置兑换汇率，选择目标货币
4. 点击 **执行兑换**

---

### 修改汇率

1. 进入 **⚙️ 汇率设置** 页面
2. 选择货币，输入新汇率（1 AUD = ?）
3. 点击 **保存汇率**

---

## 三、查看报表

### 客户流水

进入 **📝 客户流水** 页面，选择客户（可按货币筛选），可查看：

- 累计交出 / 收回金额及净变动
- 该客户所有目标账号
- 完整进出账明细及按货币统计

### 账号余额

进入 **🏦 账号余额** 页面，查看各公司账号当前余额。

### 订单管理

进入 **📋 订单管理** 页面，按状态筛选查看所有订单。

---

## 四、注意事项

> **数据安全**：系统数据存储在浏览器本地（LocalStorage），换浏览器或清除缓存会导致数据丢失，请定期截图或手动备份重要数据。

- 订单状态**不可回退**，操作前请仔细确认
- 如需添加 AUD / USD / CNY / EUR 以外的货币，需修改源码
- 当前版本无内置导出功能
