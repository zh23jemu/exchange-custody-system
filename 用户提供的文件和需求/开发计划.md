# 开发计划

---

## 当前版本功能（v1.0）

- [x] 客户、中转商、公司账号、目标账号管理
- [x] 现金订单流程（待处理 → 交中转商 → 在公司账号 → 已完成）
- [x] 直接转账订单流程（在公司账号 → 已完成）
- [x] 多币种支持（AUD / USD / CNY / EUR）
- [x] 公司账号间货币兑换
- [x] AUD 基准汇率设置
- [x] 客户流水报表（按客户、按货币）
- [x] 公司账号余额查看

---

## 待优化 / 已知问题

| 问题           | 优先级  | 说明                   |
| ------------ | ---- | -------------------- |
| 数据无导出功能      | 🔴 高 | 目前只能截图，无法导出 CSV/JSON |
| 数据无云同步       | 🔴 高 | LocalStorage 清缓存即丢失  |
| 订单不可回退       | 🟡 中 | 状态只能前进，误操作无法撤销       |
| 汇率仅支持 AUD 基准 | 🟡 中 | 非 AUD 货币间兑换精度有限      |
| 无权限/密码保护     | 🟡 中 | 任何人打开链接均可操作          |
| 货币种类固定       | 🟢 低 | 需修改源码才能添加新货币         |
